All Services Overview
🖥️ Frontend
Vue.js application logging user interactions and displaying recommendations. Stores likes in localStorage.
📱 Client Backend
Python server acting as middleware. Manages user profiles, bridges Frontend requests to Engine, validates data, and ingests events.
⚙️ Engine Backend
Python server with ML embeddings and recommendations. Indexes videos, stores events, computes similarity scores.
📊 Data Types
  • VideoRow - Video metadata (id, uuid, title, channel, thumbnail, url)
  • ChannelRow - Channel metadata (name, followers, avatar, videos_count)
  • InteractionEvent - User action (Like, UndoLike, View with timestamp and actor)
